# Ubuntu 环境

## Oh My Zsh

{% embed url="https://ohmyz.sh" caption="Oh My Zsh" %}

在 Ubuntu 下安装 oh my zsh 直接执行下面的命令即可：

```bash
sudo apt install -y git zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
```
